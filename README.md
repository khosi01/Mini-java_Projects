# JAVA-MINI-PROJECTS 

A collection of Java programs and logic notes built while mastering the fundamentals of Object-orientated programming.

### About this Repo👇

I started this journey to bridge the gap between classroom theory and real-world application. Most of these projects were built following Bro Code's legendary 12-hour roadmap. My goal is to master the Java stack to secure a junior developer role or internship while completing my 3rd year.

### 🗃️Project Structure:

I have organized this repository into Notes (syntax and concepts) and Practice Projects.

🏦**Banking program** → A simple console program that introduces the usage of methods and loops

#### **📋Core Concepts (Notes)**
⁕ **Phase 1: The Basics (Hours 0-4)**

Fundamentals: Variables, User Input, Arithmetic, Math Class.
Control Flow: If-statements, Switches, Ternary Operators.
Iteration: While, For, and Nested Loops.

⁕ **Phase 2: Data & Methods (Hours 4-6) - CURRENT FOCUS**

Methods: Declaration, Parameters, and Overloading.
Variable Scope: Local vs. Global variables.
Arrays: 1D & 2D Arrays, Array searching, and String manipulation.

⁕ **Phase 3: The OOP Shift (Hours 6-12) - UP NEXT**

Objects & Classes: Constructors, Static keyword, Inheritance.
Advanced Logic: Polymorphism, Abstraction, Interfaces.
Data Management: File I/O (Read/Write), HashMaps, and Exception Handling.

**An important lesson I have Learned:**

I used to struggle with variable initialization. Now, I use the Safe-Start Rule: declaring variables with default values (0.0, "", false) at the start of my logic blocks to ensure clean compilation.

**My Learning Philosophy🥸**

"I break down programs into logical steps using comments first. If I can't explain the logic in simple English, I don't write the code."
